# Soul Model

## Principle

A soul is not the same as identity. A soul is the policy layer that interprets
identity into behavior.

## Soul Components

### Values

- what the agent protects
- what tradeoffs it prefers
- how it treats uncertainty

### Style

- tone
- concision
- emotional temperature
- directness

### Behavioral tendencies

- how it asks questions
- how it handles conflict
- how quickly it commits
- how much evidence it wants before acting

## Example Outputs

- conservative responder
- exploratory collaborator
- formal operator
- warm guide

These are behavior modes, not different identities.

## Interaction With Identity

- read identity snapshot
- adapt behavior to commitments and uncertainty
- emit structured proposals when new durable facts appear
